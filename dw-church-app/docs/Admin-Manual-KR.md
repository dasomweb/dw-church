## 1. 교회 등록 및 로그인

### 교회 등록 (최초 1회)

1. DW Church 홈페이지에서 **"시작하기"** 버튼을 클릭합니다.
2. 다음 정보를 입력합니다:
   - **교회 이름**: 교회의 공식 명칭 (예: 다솜교회)
   - **슬러그**: 웹 주소에 사용될 고유 식별자 (예: `dasom` -> `dasom.dw-church.app`)
   - **관리자 이름**: 담당자 이름
   - **이메일**: 로그인에 사용할 이메일 주소
   - **비밀번호**: 8자 이상 권장
3. **"등록"** 버튼을 클릭하면 교회 사이트가 생성됩니다.
4. 자동으로 로그인되어 대시보드로 이동합니다.

### 로그인

1. `https://your-slug.dw-church.app/admin` 에 접속합니다.
2. 등록한 이메일과 비밀번호를 입력합니다.
3. **"로그인"** 버튼을 클릭합니다.

### 비밀번호 찾기

1. 로그인 화면에서 **"비밀번호 찾기"** 를 클릭합니다.
2. 등록한 이메일 주소를 입력합니다.
3. 이메일로 전송된 링크를 클릭하여 새 비밀번호를 설정합니다.

## 2. 대시보드 사용법

대시보드는 관리자 화면의 메인 페이지입니다.

### 대시보드 구성

- **좌측 사이드바**: 모든 관리 메뉴에 접근할 수 있는 네비게이션
  - 설교, 주보, 앨범, 교역자, 연혁, 페이지, 메뉴, 테마, 설정 등
- **상단 헤더**: 교회 이름, 사이트 보기 링크, 사용자 프로필
- **메인 영역**: 선택한 메뉴의 콘텐츠 관리 화면

### 사이트 미리보기

상단 헤더의 **"사이트 보기"** 링크를 클릭하면 방문자에게 보이는 실제 사이트를 새 탭에서 확인할 수 있습니다.

## 3. 설교 관리

### 설교 목록 보기

좌측 메뉴에서 **"설교"** 를 클릭하면 등록된 설교 목록이 표시됩니다.

- 검색: 상단 검색창에서 제목, 설교자 등으로 검색
- 필터: 카테고리, 상태(게시됨/임시저장/보관)로 필터링
- 정렬: 날짜, 제목 등으로 정렬 가능

### 설교 등록

1. 목록 상단의 **"새 설교"** 버튼을 클릭합니다.
2. 다음 항목을 입력합니다:
   - **제목** (필수): 설교 제목
   - **YouTube URL**: 설교 영상 링크
   - **성경 구절**: 본문 말씀 (예: 요한복음 3:16)
   - **설교자**: 설교자 이름
   - **날짜**: 설교 날짜
   - **썸네일**: 목록에 표시될 대표 이미지 (업로드 또는 URL 입력)
   - **카테고리**: 주일예배, 수요예배, 금요기도회 등
   - **상태**: 게시됨(published) 또는 임시저장(draft)
3. **"저장"** 버튼을 클릭합니다.

### 설교 수정

1. 목록에서 수정할 설교를 클릭합니다.
2. 원하는 항목을 수정합니다.
3. **"저장"** 버튼을 클릭합니다.

### 설교 삭제

1. 목록에서 삭제할 설교의 **삭제** 버튼을 클릭합니다.
2. 확인 대화상자에서 **"삭제"** 를 선택합니다.

> 삭제된 설교는 복구할 수 없으니 주의하세요.

## 4. 주보 관리

### 주보 등록

1. 좌측 메뉴에서 **"주보"** 를 클릭합니다.
2. **"새 주보"** 버튼을 클릭합니다.
3. 다음 항목을 입력합니다:
   - **제목** (필수): 주보 제목 (예: 2025년 1월 첫째주 주보)
   - **날짜**: 주보 날짜
   - **PDF 파일**: 주보 PDF 파일 업로드
   - **이미지**: 주보 이미지 (여러 장 업로드 가능)
   - **썸네일**: 목록에 표시될 대표 이미지
   - **상태**: 게시됨 또는 임시저장
4. **"저장"** 버튼을 클릭합니다.

### 주보 수정/삭제

설교 관리와 동일한 방식으로 수정 및 삭제가 가능합니다.

## 5. 앨범 관리

### 앨범 등록

1. 좌측 메뉴에서 **"앨범"** 을 클릭합니다.
2. **"새 앨범"** 버튼을 클릭합니다.
3. 다음 항목을 입력합니다:
   - **제목** (필수): 앨범 제목 (예: 2024년 성탄절 행사)
   - **이미지**: 여러 장의 사진을 한 번에 업로드할 수 있습니다
   - **YouTube URL**: 관련 영상이 있는 경우
   - **썸네일**: 대표 이미지 (미설정 시 첫 번째 이미지가 사용됨)
   - **카테고리**: 앨범 분류
   - **상태**: 게시됨 또는 임시저장
4. **"저장"** 버튼을 클릭합니다.

### 이미지 관리

- **추가**: 이미지 영역에서 파일을 드래그 앤 드롭하거나 **"이미지 추가"** 버튼 클릭
- **순서 변경**: 이미지를 드래그하여 순서를 변경
- **삭제**: 각 이미지의 X 버튼을 클릭하여 개별 삭제

## 6. 교역자 관리

### 교역자 등록

1. 좌측 메뉴에서 **"교역자"** 를 클릭합니다.
2. **"새 교역자"** 버튼을 클릭합니다.
3. 다음 항목을 입력합니다:
   - **이름** (필수): 교역자 이름
   - **직분**: 담임목사, 부목사, 전도사, 사모 등
   - **부서**: 소속 부서
   - **이메일**: 연락처 이메일
   - **전화번호**: 연락처 전화번호
   - **소개글**: 간단한 자기소개
   - **사진**: 프로필 사진 업로드
   - **SNS 링크**: Facebook, Instagram, YouTube URL
   - **활성 상태**: 현재 재직 중인지 여부
4. **"저장"** 버튼을 클릭합니다.

### 순서 변경

교역자 목록에서 드래그 앤 드롭으로 표시 순서를 변경할 수 있습니다. 변경된 순서는 자동으로 저장됩니다.

### 비활성화

퇴직하거나 일시적으로 표시하지 않을 교역자는 **"활성 상태"** 를 끄면 사이트에서 숨겨집니다 (삭제하지 않아도 됩니다).

## 7. 교회 연혁 관리

### 연혁 구조

연혁은 **연도** 단위로 관리됩니다. 각 연도 안에 여러 개의 **항목** (월/일 + 내용)을 추가할 수 있습니다.

### 연혁 등록

1. 좌측 메뉴에서 **"연혁"** 을 클릭합니다.
2. **"새 연도 추가"** 버튼을 클릭합니다.
3. **연도**를 입력합니다 (예: 2024).
4. 해당 연도의 항목을 추가합니다:
   - **월**: 1~12
   - **일**: (선택) 1~31
   - **내용**: 연혁 항목 설명
   - **사진**: (선택) 관련 사진
5. **"항목 추가"** 버튼으로 여러 항목을 추가합니다.
6. **"저장"** 버튼을 클릭합니다.

### 연혁 수정

1. 연도 목록에서 수정할 연도를 클릭합니다.
2. 항목을 추가, 수정 또는 삭제합니다.
3. **"저장"** 버튼을 클릭합니다.

## 8. 페이지 편집

### 페이지란?

페이지는 교회 사이트의 각 화면을 구성합니다. 홈페이지, 소개 페이지, 예배 안내 페이지 등을 만들 수 있습니다.

### 페이지 목록

좌측 메뉴에서 **"페이지"** 를 클릭하면 모든 페이지 목록이 표시됩니다.

### 새 페이지 만들기

1. **"새 페이지"** 버튼을 클릭합니다.
2. 다음을 입력합니다:
   - **제목**: 페이지 제목
   - **슬러그**: URL에 사용될 경로 (예: `about` -> `/about`)
   - **홈 페이지 여부**: 이 페이지를 메인 페이지로 설정
   - **상태**: 게시됨 또는 임시저장
3. **"저장"** 을 클릭합니다.

### 블록 추가

페이지는 **블록(섹션)** 을 조합하여 구성합니다. 사용 가능한 블록 종류:

| 블록 | 설명 |
|------|------|
| 히어로 배너 | 전체 너비 배너 이미지 |
| 텍스트+이미지 | 텍스트와 이미지를 나란히 배치 |
| 텍스트만 | 텍스트 콘텐츠 영역 |
| 이미지 갤러리 | 여러 이미지를 격자형으로 배치 |
| 동영상 | YouTube 등 영상 임베드 |
| 구분선 | 섹션 간 구분선 |
| 최근 설교 | 최근 설교 자동 표시 |
| 최근 주보 | 최근 주보 자동 표시 |
| 앨범 갤러리 | 앨범 사진 갤러리 |
| 교역자 목록 | 교역자 프로필 카드 |
| 교회 연혁 | 타임라인 형태의 연혁 |
| 행사 목록 | 교회 행사/이벤트 카드 |
| 예배 안내 | 예배 시간표 |
| 위치/지도 | 교회 위치 지도 |
| 연락처 정보 | 교회 연락처 표시 |
| 새가족 안내 | 새가족을 위한 안내 정보 |
| 2단 레이아웃 | 콘텐츠를 2열로 분할 |
| 3단 레이아웃 | 콘텐츠를 3열로 분할 |
| 탭 | 탭으로 콘텐츠 전환 |
| 아코디언 | 접었다 펼 수 있는 FAQ 형태 |

### 블록 편집

1. 페이지 편집 화면에서 블록을 클릭합니다.
2. 블록별 설정 패널이 열립니다.
3. 내용을 수정한 후 저장합니다.

### 블록 순서 변경

블록을 드래그 앤 드롭하여 순서를 변경할 수 있습니다.

### 블록 삭제

블록의 **삭제** 버튼을 클릭하고 확인합니다.

### 블록 숨기기

삭제하지 않고 일시적으로 숨기려면 블록의 **"보이기/숨기기"** 토글을 사용합니다.

## 9. 메뉴 관리

### 메뉴 구조

메뉴는 사이트 상단 네비게이션을 구성합니다. 1단계 메뉴 아래에 하위 메뉴를 추가할 수 있습니다.

### 메뉴 항목 추가

1. 좌측 메뉴에서 **"메뉴"** 를 클릭합니다.
2. **"메뉴 항목 추가"** 버튼을 클릭합니다.
3. 다음을 입력합니다:
   - **레이블**: 메뉴에 표시될 텍스트 (예: 교회소개, 예배안내)
   - **연결 페이지**: 이 메뉴를 클릭했을 때 이동할 페이지 선택
   - **외부 URL**: 페이지 대신 외부 링크를 연결 (선택)
   - **상위 메뉴**: 하위 메뉴로 만들 경우 상위 메뉴 선택
   - **표시 여부**: 메뉴를 보이게/숨기게 설정
4. **"저장"** 을 클릭합니다.

### 메뉴 순서 변경

드래그 앤 드롭으로 메뉴 순서를 변경합니다. 변경 후 자동 저장됩니다.

### 메뉴 삭제

메뉴 항목의 **삭제** 버튼을 클릭합니다. 하위 메뉴가 있는 경우 함께 삭제되니 주의하세요.

## 10. 테마 설정

### 색상 변경

1. 좌측 메뉴에서 **"테마"** 를 클릭합니다.
2. 다음 색상을 설정합니다:
   - **주요 색상 (Primary)**: 버튼, 링크 등에 사용되는 메인 색상
   - **보조 색상 (Secondary)**: 부가 요소에 사용되는 색상
   - **강조 색상 (Accent)**: 특별히 강조할 요소의 색상
   - **배경 색상 (Background)**: 페이지 전체 배경색
   - **표면 색상 (Surface)**: 카드, 패널 등의 배경색
   - **텍스트 색상 (Text)**: 본문 텍스트 색상
3. 색상 선택기를 사용하거나 HEX 코드를 직접 입력합니다.
4. **"저장"** 을 클릭합니다.

### 폰트 변경

- **제목 폰트**: 제목에 사용할 폰트 (예: Pretendard, Noto Sans KR)
- **본문 폰트**: 일반 텍스트에 사용할 폰트

### 커스텀 CSS

고급 사용자를 위해 **"커스텀 CSS"** 입력란에 추가 CSS 코드를 작성할 수 있습니다. 이 CSS는 사이트 전체에 적용됩니다.

## 11. 설정

### 교회 정보

좌측 메뉴에서 **"설정"** 을 클릭합니다.

| 항목 | 설명 |
|------|------|
| 교회 이름 | 사이트에 표시되는 교회 이름 |
| 주소 | 교회 주소 |
| 전화번호 | 대표 전화번호 |
| 이메일 | 대표 이메일 주소 |
| 웹사이트 | 교회 홈페이지 URL |

### SNS 연동

각 SNS 계정의 URL을 입력하면 사이트 하단 등에 아이콘이 표시됩니다.

| 항목 | 예시 |
|------|------|
| YouTube | `https://youtube.com/@mychurch` |
| Instagram | `https://instagram.com/mychurch` |
| Facebook | `https://facebook.com/mychurch` |
| LinkedIn | `https://linkedin.com/company/mychurch` |
| TikTok | `https://tiktok.com/@mychurch` |
| 카카오톡 | 카카오톡 오픈채팅 URL |
| 카카오톡 채널 | 카카오톡 채널 URL |

모든 항목은 선택 사항입니다. 입력하지 않으면 해당 SNS 아이콘은 표시되지 않습니다.

## 12. 사용자 관리

### 사용자 역할

| 역할 | 설명 |
|------|------|
| **소유자 (Owner)** | 모든 권한. 교회 등록자에게 자동 부여. 교회당 1명. |
| **관리자 (Admin)** | 모든 콘텐츠 관리 + 사용자 초대 가능 |
| **편집자 (Editor)** | 콘텐츠 등록/수정만 가능. 설정 및 사용자 관리 불가. |

### 팀원 초대

1. 좌측 메뉴에서 **"설정 > 사용자"** 를 클릭합니다.
2. **"팀원 초대"** 버튼을 클릭합니다.
3. 다음을 입력합니다:
   - **이메일**: 초대할 사람의 이메일 주소
   - **이름**: 표시될 이름
   - **역할**: 관리자(Admin) 또는 편집자(Editor)
4. **"초대"** 버튼을 클릭합니다.
5. 초대된 사람에게 이메일이 발송되며, 링크를 통해 비밀번호를 설정하고 참여합니다.

### 팀원 제거

1. 사용자 목록에서 제거할 사용자의 **"제거"** 버튼을 클릭합니다.
2. 확인 대화상자에서 **"제거"** 를 선택합니다.

> 소유자(Owner) 계정은 제거할 수 없습니다.

## 자주 묻는 질문

### Q: 사이트 주소(슬러그)를 변경할 수 있나요?

등록 후 슬러그 변경은 현재 지원되지 않습니다. 변경이 필요한 경우 관리자에게 문의해 주세요.

### Q: 이미지 파일 크기 제한이 있나요?

개별 파일당 최대 10MB까지 업로드할 수 있습니다. 웹 최적화를 위해 2MB 이하의 이미지를 권장합니다.

### Q: 게시 상태의 차이점은 무엇인가요?

- **게시됨 (Published)**: 사이트 방문자에게 공개됩니다.
- **임시저장 (Draft)**: 관리자만 볼 수 있으며, 사이트에 표시되지 않습니다.
- **보관 (Archived)**: 게시 중지된 콘텐츠. 삭제하지 않고 보관합니다.

### Q: 여러 명이 동시에 관리할 수 있나요?

네, 팀원을 초대하여 여러 명이 동시에 콘텐츠를 관리할 수 있습니다. 각 사용자는 고유한 이메일과 비밀번호로 로그인합니다.

### Q: 데이터 백업은 어떻게 하나요?

DW Church SaaS에서 자동으로 데이터를 백업합니다. 별도의 백업 조치가 필요하지 않습니다.
