## ② 한 문단 소개 (헤드라인 바로 아래)

TRUE LIGHT는 한인 교회를 위한 **온라인 사역 전문 솔루션**입니다. 일반 홈페이지
만들기 도구와 달리, 설교·주보·교역자·새가족·목장처럼 **교회에만 필요한 기능**이
처음부터 들어 있습니다. 어려운 기술과 디자인은 전문성으로 갖춰진 채 시작하고,
교회는 글과 사진만 채우며 본연의 사역에 집중하면 됩니다.

---

## ③ 왜 TRUE LIGHT인가 — 4가지 (아이콘 + 한 줄)

- ⛪ **교회 전용** — 설교·주보·목장·새가족 등 교회에 꼭 맞는 기능이 기본 탑재
- 🎨 **전문성 내장** — 전문가 수준의 디자인과 구축이 처음부터 완성된 채로
- 🇰🇷 **한인 교회 맞춤** — 한국어·영어, 한인 교회 문화에 맞춘 구성
- 🔐 **교회 자료는 교회 것** — 언제든 전체 자료를 내려받을 수 있음(안심)

도메인 연결 설정(도메인은 교회 직접 구입) · 오픈 전 점검. → *이후에는 매달(또는 매년)
이용료만.*

---

## ⑤ 시작 방법 (네 단계)

1. **신청** — 교회에 맞는 단계 선택 + 간단한 개발 신청서 작성
2. **준비** — 교회에 맞는 디자인 적용, (기존 사이트가 있으면) 내용 이전 → 홈페이지 완성
3. **내용 채우기** — 설교·주보·사진을 쉬운 관리 화면에서 등록
4. **오픈** — 교회 주소 연결 후 성도에게 공개

> 신청서 검토 후 결제 안내를 보내드리고, 결제가 확인되면 제작을 시작합니다.

## (영업 보조) 자주 받는 질문 — 한 줄 답

- **Q. 우리가 직접 다 만들어야 하나요?** → 아니요. 디자인·구축은 완성된 채 시작하고, 교회는 글·사진만 등록합니다.
- **Q. 기존 홈페이지가 있어요.** → 신청 시 주소를 주시면 기존 내용을 그대로 옮겨드립니다.
- **Q. 매달 비용 외에 셋업비는 왜 있나요?** → 초기 구축(디자인·이전)에 들어가는 1회 비용입니다. 원래 제작 업체에 맡기면 훨씬 큽니다.
- **Q. 교인 명단 관리·헌금 결제도 되나요?** → 현재는 **온라인 사역(홈페이지·게시판·새가족·목장 등)** 중심입니다. 헌금은 *방법 안내* 페이지를 제공합니다.
- **Q. 나중에 그만두면 자료는요?** → 전체 자료를 언제든 내려받을 수 있습니다.

> ⚠️ 영업 가드레일: 없는 기능(교인 DB·헌금 결제 처리·방문자 분석)은 절대 약속하지 말 것. 디자인 테마는 6종.
