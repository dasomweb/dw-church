## 1. 교회주보 (Bulletin)

### 1.1 새 주보 작성하기

#### 접근 방법
1. WordPress 관리자 페이지에 로그인합니다.
2. 왼쪽 메뉴에서 **"교회주보"** 또는 **"Bulletins"**를 클릭합니다.
3. 상단의 **"새로 추가"** 또는 **"Add New"** 버튼을 클릭합니다.

#### 필수 필드 입력

**주보 정보 메타박스**에서 다음 필드를 입력합니다:

##### 주보 날짜 (필수)
- **필드 위치**: 주보 정보 메타박스 상단
- **입력 방법**: 날짜 선택기에서 날짜를 선택하거나 직접 입력 (YYYY-MM-DD 형식)
- **설명**: 이 주보가 해당하는 날짜를 선택합니다.
- **자동 기능**: 날짜를 입력하면 포스트 제목이 자동으로 생성됩니다.
  - 예: "2025년 12월 25일 교회주보"

##### 주보 PDF (선택)
- **필드 위치**: 주보 날짜 아래
- **입력 방법**:
  1. **"PDF 업로드"** 버튼을 클릭합니다.
  2. 미디어 라이브러리에서 PDF 파일을 선택하거나 새로 업로드합니다.
  3. 선택한 PDF 파일이 자동으로 연결됩니다.
- **설명**: 주보 PDF 파일을 업로드하거나 기존 미디어 라이브러리에서 선택할 수 있습니다.
- **확인 방법**: PDF가 업로드되면 **"현재 PDF 보기"** 링크가 표시됩니다.

##### 주보 이미지 (선택)
- **필드 위치**: 주보 PDF 아래
- **입력 방법**:
  1. **"이미지 업로드/선택"** 버튼을 클릭합니다.
  2. 미디어 라이브러리에서 이미지를 선택하거나 새로 업로드합니다.
  3. 여러 이미지를 선택할 수 있습니다.
  4. 선택한 이미지는 썸네일로 표시됩니다.
- **설명**: 주보와 관련된 이미지를 추가할 수 있습니다.
- **자동 기능**: 첫 번째 이미지가 자동으로 대표 이미지(Featured Image)로 설정됩니다.

#### 저장 및 발행

1. **임시 저장**: 
   - 오른쪽 상단의 **"초안 저장"** 버튼을 클릭하면 나중에 수정할 수 있도록 저장됩니다.
   - 포스트 상태는 "초안(Draft)"으로 유지됩니다.

2. **발행하기**:
   - 모든 필드를 입력한 후 오른쪽 상단의 **"발행"** 또는 **"Publish"** 버튼을 클릭합니다.
   - 포스트가 즉시 사이트에 공개됩니다.

3. **예약 발행**:
   - **"발행"** 버튼 옆의 **"즉시"**를 클릭하여 날짜와 시간을 설정할 수 있습니다.
   - 지정한 시간에 자동으로 발행됩니다.

### 1.2 주보 업데이트하기

#### 기존 주보 수정

1. **접근 방법**:
   - **"교회주보"** 메뉴에서 수정할 주보를 클릭합니다.
   - 또는 주보 목록에서 **"편집"** 링크를 클릭합니다.

2. **필드 수정**:
   - 주보 날짜, PDF, 이미지 등 모든 필드를 수정할 수 있습니다.
   - 날짜를 변경하면 포스트 제목이 자동으로 업데이트됩니다.

3. **저장**:
   - 수정 후 오른쪽 상단의 **"업데이트"** 버튼을 클릭합니다.
   - 변경사항이 즉시 반영됩니다.

#### 주보 삭제하기

1. 주보 편집 페이지에서 오른쪽 상단의 **"휴지통으로 이동"** 또는 **"Move to Trash"** 버튼을 클릭합니다.
2. 또는 주보 목록에서 주보를 선택하고 상단의 **"휴지통으로 이동"**을 선택합니다.

## 2. 설교 (Sermon)

### 2.1 새 설교 작성하기

#### 접근 방법
1. WordPress 관리자 페이지에 로그인합니다.
2. 왼쪽 메뉴에서 **"설교"** 또는 **"Sermons"**를 클릭합니다.
3. 상단의 **"새로 추가"** 또는 **"Add New"** 버튼을 클릭합니다.

#### 필수 필드 입력

**설교 정보 메타박스**에서 다음 필드를 입력합니다:

##### 설교 제목 (필수)
- **필드 위치**: 설교 정보 메타박스 상단
- **입력 방법**: 텍스트 입력란에 설교 제목을 입력합니다.
- **설명**: 설교의 제목을 입력합니다.
- **자동 기능**: 입력한 제목이 포스트 제목으로 자동 설정됩니다.

##### 성경구절 (선택)
- **필드 위치**: 설교 제목 아래
- **입력 방법**: 텍스트 입력란에 성경구절을 입력합니다.
- **예시**: "요한복음 3:16", "마태복음 5:1-12" 등

##### 설교자 (선택)
- **필드 위치**: 성경구절 아래
- **입력 방법**: 드롭다운 메뉴에서 설교자를 선택합니다.
- **설명**: 
  - 설교자 목록은 **"교회관리 → 대시보드 → 설교자 관리"**에서 관리할 수 있습니다.
  - 기본값으로 "담임목사"가 설정되어 있을 수 있습니다.

##### YouTube URL (선택)
- **필드 위치**: 설교자 아래
- **입력 방법**: YouTube 비디오 URL을 입력합니다.
- **예시**: 
  - `https://www.youtube.com/watch?v=VIDEO_ID`
  - `https://youtu.be/VIDEO_ID`

##### 설교 일자 (선택)
- **필드 위치**: YouTube URL 아래
- **입력 방법**: 날짜 선택기에서 날짜를 선택하거나 직접 입력합니다.
- **설명**: 설교가 진행된 날짜를 입력합니다.

##### YouTube 썸네일 (선택)
- **필드 위치**: 설교 일자 아래
- **입력 방법**:
  1. **"썸네일 업로드/선택"** 버튼을 클릭하여 직접 업로드
  2. 또는 **"YouTube 썸네일 불러오기"** 버튼을 클릭하여 YouTube에서 자동으로 가져오기
  3. **"썸네일 삭제"** 버튼으로 삭제 가능
- **설명**: 
  - YouTube URL을 입력한 경우, **"YouTube 썸네일 불러오기"** 버튼을 사용하면 자동으로 썸네일을 가져올 수 있습니다.
  - 저장 시 썸네일이 대표 이미지(Featured Image)로 자동 설정됩니다.

#### 저장 및 발행

1. **임시 저장**: **"초안 저장"** 버튼 클릭
2. **발행하기**: **"발행"** 버튼 클릭
3. **예약 발행**: 발행 시간을 설정하여 예약 가능

### 2.2 설교 업데이트하기

#### 기존 설교 수정

1. **접근 방법**: **"설교"** 메뉴에서 수정할 설교를 클릭하거나 목록에서 **"편집"** 링크 클릭
2. **필드 수정**: 모든 필드를 수정할 수 있습니다.
3. **저장**: **"업데이트"** 버튼 클릭

#### 설교 삭제하기

1. 설교 편집 페이지에서 **"휴지통으로 이동"** 버튼 클릭
2. 또는 목록에서 여러 설교를 선택하여 일괄 삭제 가능

## 3. 교회앨범 (Album)

### 3.1 새 앨범 작성하기

#### 접근 방법
1. WordPress 관리자 페이지에 로그인합니다.
2. 왼쪽 메뉴에서 **"교회앨범"** 또는 **"Albums"**를 클릭합니다.
3. 상단의 **"새로 추가"** 또는 **"Add New"** 버튼을 클릭합니다.

#### 필수 필드 입력

**앨범 정보 메타박스**에서 다음 필드를 입력합니다:

##### 앨범 이미지 (선택)
- **필드 위치**: 앨범 정보 메타박스 상단
- **입력 방법**:
  1. **"이미지 업로드/선택"** 버튼을 클릭합니다.
  2. 미디어 라이브러리 창이 열립니다.
  3. 여러 이미지를 선택할 수 있습니다 (최대 15개).
  4. 선택한 이미지는 썸네일로 표시되며, 각 썸네일에서 **"×"** 버튼으로 삭제할 수 있습니다.
- **권장사항**: 
  - 이미지 개수는 **10-15개 정도로 제한**하는 것을 권장합니다.
  - 이미지가 많을 경우 갤러리 로딩 시 속도 문제가 발생할 수 있습니다.
- **이미지 개수 제한**: 
  - **15개를 초과하면 저장이 되지 않습니다.**
  - 15개 초과 시 경고 메시지가 표시되고, 저장 버튼을 눌러도 이미지가 저장되지 않습니다.
- **자동 기능**: 
  - 첫 번째 이미지가 자동으로 대표 이미지(Featured Image)로 설정됩니다.
  - 단, YouTube 썸네일이 설정되어 있으면 YouTube 썸네일이 우선됩니다.

##### YouTube URL (선택)
- **필드 위치**: 앨범 이미지 아래
- **입력 방법**: YouTube 비디오 URL을 입력합니다.
- **설명**: 앨범과 관련된 YouTube 비디오가 있는 경우 입력합니다.

##### YouTube 썸네일 (선택)
- **필드 위치**: YouTube URL 아래
- **입력 방법**:
  1. **"썸네일 업로드/선택"**: 직접 이미지 업로드
  2. **"YouTube 썸네일 불러오기"**: YouTube URL에서 자동으로 썸네일 가져오기
  3. **"썸네일 삭제"**: 삭제
- **설명**: 
  - YouTube URL을 입력한 경우, **"YouTube 썸네일 불러오기"** 버튼을 사용하면 자동으로 썸네일을 가져올 수 있습니다.
  - YouTube 썸네일이 설정되면 대표 이미지로 우선 사용됩니다.

#### 이미지 선택 팁

1. **이미지 순서 변경**:
   - 현재는 이미지를 선택한 순서대로 저장됩니다.
   - 순서를 변경하려면 이미지를 삭제하고 다시 선택 순서대로 추가하세요.

2. **이미지 삭제**:
   - 각 썸네일의 **"×"** 버튼을 클릭하면 즉시 삭제됩니다.
   - 삭제 후 **"업데이트"** 버튼을 클릭하여 저장해야 합니다.

3. **이미지 개수 확인**:
   - 현재 선택된 이미지 개수가 **"현재 N개 이미지"**로 표시됩니다.

#### 저장 및 발행

1. **임시 저장**: **"초안 저장"** 버튼 클릭
2. **발행하기**: **"발행"** 버튼 클릭
3. **주의사항**: 
   - 이미지가 15개를 초과하면 저장이 되지 않습니다.
   - 저장 전에 이미지 개수를 확인하세요.

### 3.2 앨범 업데이트하기

#### 기존 앨범 수정

1. **접근 방법**: **"교회앨범"** 메뉴에서 수정할 앨범을 클릭하거나 목록에서 **"편집"** 링크 클릭

2. **이미지 추가/삭제**:
   - **"이미지 업로드/선택"** 버튼으로 추가 이미지 선택
   - 각 썸네일의 **"×"** 버튼으로 이미지 삭제
   - **15개 초과 시 저장되지 않음**

3. **필드 수정**: YouTube URL, 썸네일 등 모든 필드 수정 가능

4. **저장**: **"업데이트"** 버튼 클릭

#### 앨범 삭제하기

1. 앨범 편집 페이지에서 **"휴지통으로 이동"** 버튼 클릭
2. 또는 목록에서 여러 앨범을 선택하여 일괄 삭제 가능

## 4. 배너 (Banner)

### 4.1 새 배너 작성하기

#### 접근 방법
1. WordPress 관리자 페이지에 로그인합니다.
2. 왼쪽 메뉴에서 **"배너"** 또는 **"Banners"**를 클릭합니다.
3. 상단의 **"새로 추가"** 또는 **"Add New"** 버튼을 클릭합니다.

#### 배너 유형

배너는 **메인 배너**와 **서브 배너** 두 가지 유형이 있습니다:

- **메인 배너**: PC용(1920px) + 모바일용(720px) 이미지 사용
- **서브 배너**: 1024px 고정폭 이미지 사용 (비율 선택 가능: 16:9, 4:3, 1:1)

#### 필수 필드 입력

**배너 정보 메타박스**에서 다음 필드를 입력합니다:

##### 메인 배너 필드

**PC용 배너 이미지 (1920px)**
- **필드 위치**: 배너 정보 메타박스 상단
- **입력 방법**:
  1. **"PC용 이미지 업로드"** 버튼을 클릭합니다.
  2. 미디어 라이브러리에서 이미지를 선택하거나 새로 업로드합니다.
  3. **"이미지 삭제"** 버튼으로 삭제 가능합니다.
- **권장 크기**: 가로 1920px

**모바일용 배너 이미지 (720px)**
- **필드 위치**: PC용 이미지 아래
- **입력 방법**: **"모바일용 이미지 업로드"** 버튼을 클릭합니다.
- **권장 크기**: 가로 720px

##### 서브 배너 필드

**서브 배너 이미지 (1024px)**
- **필드 위치**: 메인 배너 필드 아래
- **입력 방법**: **"서브 배너 이미지 업로드"** 버튼을 클릭합니다.
- **권장 크기**: 가로 1024px

**서브 배너 비율**
- **필드 위치**: 서브 배너 이미지 아래
- **선택 옵션**: 16:9, 4:3, 1:1
- **기본값**: 16:9

##### 링크 설정

**링크 URL**
- **필드 위치**: 배너 이미지 필드 아래
- **입력 방법**: 클릭 시 이동할 URL을 입력합니다.
- **예시**: `https://example.com/page`

**링크 타겟**
- **필드 위치**: 링크 URL 옆
- **선택 옵션**: 
  - **같은 창에서 열기** (_self)
  - **새 창에서 열기** (_blank)
- **기본값**: 같은 창에서 열기

##### 날짜 설정

**시작 날짜**
- **필드 위치**: 링크 설정 아래
- **입력 방법**: 날짜 선택기에서 시작 날짜를 선택합니다.
- **설명**: 배너가 표시되기 시작하는 날짜입니다.

**종료 날짜**
- **필드 위치**: 시작 날짜 아래
- **입력 방법**: 날짜 선택기에서 종료 날짜를 선택합니다.
- **설명**: 배너가 표시되는 마지막 날짜입니다.

##### 텍스트 오버레이 (선택)

배경 이미지 위에 텍스트를 표시할 수 있습니다. 입력하지 않으면 이미지만 표시됩니다.

**텍스트 제목**
- **필드 위치**: 날짜 설정 아래
- **입력 방법**: 텍스트 입력란에 제목을 입력합니다.

**텍스트 부제목**
- **필드 위치**: 텍스트 제목 아래
- **입력 방법**: 텍스트 입력란에 부제목을 입력합니다.

**텍스트 설명**
- **필드 위치**: 텍스트 부제목 아래
- **입력 방법**: 텍스트 영역에 설명을 입력합니다.

**텍스트 위치**
- **필드 위치**: 텍스트 설명 아래
- **선택 옵션**: 
  - 상단-좌측 (top-left)
  - 상단-중앙 (top-center)
  - 상단-우측 (top-right)
  - 중앙-좌측 (center-left)
  - 중앙-중앙 (center-center) - **기본값**
  - 중앙-우측 (center-right)
  - 하단-좌측 (bottom-left)
  - 하단-중앙 (bottom-center)
  - 하단-우측 (bottom-right)

**텍스트 정렬**
- **필드 위치**: 텍스트 위치 아래
- **선택 옵션**: 좌측 정렬, 중앙 정렬, 우측 정렬
- **기본값**: 중앙 정렬

**텍스트 너비 (반응형)**
- **필드 위치**: 텍스트 정렬 아래
- **설정 항목**:
  - PC용 텍스트 너비 (기본값: 600px)
  - 노트북용 텍스트 너비 (기본값: 600px)
  - 태블릿용 텍스트 너비 (기본값: 500px)
  - 모바일용 텍스트 너비 (기본값: 300px)

**배경 이미지 위치 (반응형)**
- **필드 위치**: 텍스트 너비 아래
- **설정 항목**: PC, 노트북, 태블릿, 모바일별로 배경 이미지 위치 설정
- **기본값**: center center (중앙-중앙)

**콘텐츠 패딩**
- **필드 위치**: 배경 이미지 위치 아래
- **설정 항목**: 상단, 우측, 하단, 좌측 패딩 값 설정
- **기본값**: 40px

## 5. 이벤트 (Event)

### 5.1 새 이벤트 작성하기

#### 접근 방법
1. WordPress 관리자 페이지에 로그인합니다.
2. 왼쪽 메뉴에서 **"이벤트"** 또는 **"Events"**를 클릭합니다.
3. 상단의 **"새로 추가"** 또는 **"Add New"** 버튼을 클릭합니다.

#### 필수 필드 입력

**이벤트 정보 메타박스**에서 다음 필드를 입력합니다:

##### 배경 이미지 (선택)
- **필드 위치**: 이벤트 정보 메타박스 상단
- **입력 방법**:
  1. **"이미지 업로드"** 버튼을 클릭합니다.
  2. 미디어 라이브러리에서 이미지를 선택하거나 새로 업로드합니다.
  3. **"이미지 삭제"** 버튼으로 삭제 가능합니다.
- **설명**: 배경 이미지 위에 이벤트 제목, 날짜/시간, Read More 버튼이 표시됩니다.

##### 부서 (선택)
- **필드 위치**: 배경 이미지 아래
- **입력 방법**: 텍스트 입력란에 부서명을 입력합니다.
- **예시**: "청년부", "유년부", "선교부" 등
- **설명**: 이벤트를 주관하는 부서를 입력합니다.

##### 이벤트 날짜 및 시간 (선택)
- **필드 위치**: 부서 아래
- **입력 방법**: 텍스트 입력란에 날짜와 시간을 자유 형식으로 입력합니다.
- **예시**: "2025년 12월 25일 오후 3시", "2025-12-25 15:00" 등
- **설명**: 이벤트가 열리는 날짜와 시간을 입력합니다. (자유 텍스트 형식)

##### 이벤트 URL (선택)
- **필드 위치**: 이벤트 날짜 및 시간 아래
- **입력 방법**: URL 입력란에 링크를 입력합니다.
- **예시**: `https://example.com/event-page`
- **설명**: 이벤트 상세 페이지 또는 외부 링크 URL을 입력합니다.

**새창으로 열기**
- **필드 위치**: 이벤트 URL 옆 체크박스
- **설명**: 체크하면 링크가 새 창에서 열립니다.

##### 행사 설명 (선택)
- **필드 위치**: 이벤트 URL 아래
- **입력 방법**: 텍스트 영역에 설명을 입력합니다.
- **설명**: 이벤트에 대한 상세 설명을 입력합니다.

##### YouTube URL (선택)
- **필드 위치**: 행사 설명 아래
- **입력 방법**: YouTube 비디오 URL을 입력합니다.
- **예시**: `https://www.youtube.com/watch?v=VIDEO_ID`
- **설명**: YouTube 비디오 링크를 입력하면 썸네일을 자동으로 가져올 수 있습니다.

##### YouTube 썸네일 (선택)
- **필드 위치**: YouTube URL 아래
- **입력 방법**:
  1. **"썸네일 업로드/선택"**: 직접 이미지 업로드
  2. **"YouTube 썸네일 불러오기"**: YouTube URL에서 자동으로 썸네일 가져오기
  3. **"썸네일 삭제"**: 삭제
- **설명**: 
  - YouTube URL을 입력한 경우, **"YouTube 썸네일 불러오기"** 버튼을 사용하면 자동으로 썸네일을 가져올 수 있습니다.
  - 저장 시 썸네일이 대표 이미지(Featured Image)로 자동 설정됩니다.

#### 저장 및 발행

1. **임시 저장**: **"초안 저장"** 버튼 클릭
2. **발행하기**: **"발행"** 버튼 클릭
3. **예약 발행**: 발행 시간을 설정하여 예약 가능

### 5.2 이벤트 업데이트하기

#### 기존 이벤트 수정

1. **접근 방법**: **"이벤트"** 메뉴에서 수정할 이벤트를 클릭하거나 목록에서 **"편집"** 링크 클릭

2. **필드 수정**: 
   - 배경 이미지, 부서, 날짜/시간, URL, 설명, YouTube URL, 썸네일 등 모든 필드 수정 가능

3. **저장**: **"업데이트"** 버튼 클릭

#### 이벤트 삭제하기

1. 이벤트 편집 페이지에서 **"휴지통으로 이동"** 버튼 클릭
2. 또는 목록에서 여러 이벤트를 선택하여 일괄 삭제 가능

## 공통 기능

### 이미지 업로드/선택 방법

1. **"이미지 업로드"** 또는 **"이미지 업로드/선택"** 버튼을 클릭합니다.
2. 미디어 라이브러리 창이 열립니다.
3. **"파일 업로드"** 탭에서 새 파일을 업로드하거나, **"미디어 라이브러리"** 탭에서 기존 이미지를 선택합니다.
4. 여러 이미지를 선택하려면 **Ctrl 키(Windows)** 또는 **Cmd 키(Mac)**를 누른 채 이미지를 클릭합니다.
5. 선택 완료 후 **"선택"** 버튼을 클릭합니다.

### 포스트 제목 설정

- 대부분의 포스트 타입에서는 제목이 자동으로 생성되거나 메타 필드 값에 따라 업데이트됩니다.
- 필요시 상단의 제목 입력란에서 직접 수정할 수 있습니다.

### 포스트 상태

- **초안 (Draft)**: 아직 발행되지 않은 포스트
- **발행됨 (Published)**: 사이트에 공개된 포스트
- **예약됨 (Scheduled)**: 예약 발행된 포스트
- **개인용 (Private)**: 관리자만 볼 수 있는 포스트
- **비밀번호 보호 (Password Protected)**: 비밀번호로 보호된 포스트

### 포스트 카테고리 및 태그

- 일부 포스트 타입(예: 배너)은 카테고리를 사용할 수 있습니다.
- 카테고리는 포스트 타입별로 관리됩니다.

## 문제 해결

### 이미지가 저장되지 않을 때

1. **이미지 개수 확인**: 교회앨범의 경우 15개를 초과하면 저장되지 않습니다.
2. **브라우저 새로고침**: 페이지를 새로고침하고 다시 시도합니다.
3. **파일 크기 확인**: 업로드하는 파일 크기가 서버 제한을 초과하지 않는지 확인합니다.

### PDF가 업로드되지 않을 때

1. **파일 형식 확인**: PDF 파일 형식인지 확인합니다.
2. **파일 크기 확인**: 서버의 업로드 크기 제한을 확인합니다.
3. **미디어 라이브러리 확인**: 미디어 라이브러리에서 직접 업로드를 시도합니다.

### YouTube 썸네일이 불러와지지 않을 때

1. **URL 형식 확인**: 올바른 YouTube URL 형식인지 확인합니다.
2. **인터넷 연결 확인**: 서버가 YouTube에 접근할 수 있는지 확인합니다.
3. **수동 업로드**: **"썸네일 업로드/선택"** 버튼으로 수동으로 업로드합니다.
